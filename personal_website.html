Building a personal website

What I learned from this project

I learned HTML, CSS, and git at by doing this project. I also got some experience with Amazon EC2.

Why create one from scratch?

There is no shortage of templates on bootstrap and h5up.

The background

At my day job, I use the command line and shell scripts pretty regularly. I think having a good understanding of Linux shells and working on the command line is important when working on any personal projects: It's easier to visualize the pieces of a website when I have a conceptual understanding of directory trees. I use this guide from time-to-time as a reference: https://guide.bash.academy I became aware of git and amazon EC2 instances from the ruby on rails tutorial. These two tools are indispensable to me. At home, I can work on projects from my laptop. And during work lunch, I can work on projects from an Amazon EC2 instance. I can manage my projects from the git repo. Being self-taught has its challenges, but I also appreciate several things about it. I feel it has made me more discerning about the references I depend on, allows me to dictate the pace, and lets me explore as deeply as I want. https://www.atlassian.com/git/tutorials and https://developer.mozilla.org/en-US/docs/Web/HTML have been useful in answering any questions I have about git or html.
